# C Console Chat App
A simple console-based chatting application written in C, compatible with Pelles C (Windows).

## Project Files
This repository contains all source code files for the app:
- `main.c`: Entry point of the program (login menu, main logic, data loading)
- `user.c` / `user.h`: Functions for user login/registration (login, signup, load user, save user, find user)
- `friends.c` / `friends.h`: Functions for friend management (add friends, accept friend, view friends, delete friends)
- `message.c` / `message.h`: Functions for sending/receiving messages (send message, read message, delete message)
